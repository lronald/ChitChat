Class Summary

Classes

Description

DbReferences

Contains database schema constants such as database name, username, password, table names, and more.

User

Represents a single user.

Group

Represents a single group. That is, a group of users.

Message

Represents a single message sent between two people.

GroupMessage

Represents a single message sent to a group. That is, more than two people.

UserManager

Provides functionality to manipulate users (create, delete, and more).

GroupManager

Provides functionality to manipulate groups (create, delete, and more).

MessageManager

Provides functionality to manipulate messages (create, delete, and more).

GroupMessageManager

Provides functionality to manipulate group messages (create, delete, and more).

GroupMemberManager

Provides functionality to manipulate group members (create, delete, and more).

MySqlDbAdapter

Provides functionality to manipulate MySQL database. It provides a high-level of abstraction, simplifying its use.
