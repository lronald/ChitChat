Installatin of the ChitChat web application is pretty straight forward!

Requirements

The following tools & packages are required to install, host & operate ChitChat. Hence, ensure these are installed accordingly;

  • XAMPP (Apache, MySQL, PHP) - v3.2.1

  • Apache Web Server - v2.4.9

  • MySQL - v5.6.16

  • PHP - v5.5.11

The specified requirements above are provided for in all-in-one packages especially XAMPP/Apache. However, others may be used such as Wamp Server. Since XAMPP was used as the testing server for the web application, it is highly preferred.

 

  
 

MySQL Database

MySQL database is used in the project. Hence, ChitChat will operate safely on MySQL Community Edition version 5.6.16 or later.

 
 

Web Server

Apache Server is used. ChitChat will operate safely on Apache Web Server version 2.4.9 or later.
