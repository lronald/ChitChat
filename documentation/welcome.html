What is ChitChat

ChitChat is a web application for chat and convos between people, and even groups. ChitChat is a fun project that was developed by Louis Ronald, and aims to encourage continous development and input from other collaboraters. We don't know what it can be, or where we're headed. But we believe that as this software grows, its purpose and usefulness will be discovered!
