Technologies Used

ChitChat makes use of various technologies available today such as;

MySQL

MySQL is used to store all administrative and user data used in the system.

XAMPP

Although ChitChat is developed to be used by other web servers as well, XAMPP seems to be the more preferred option. XAMPP contains the Apache Web Server, and PhpMyAdmin interface for the MySQL database.

PHP

PHP is used to write the server-side scripts, handling most of the database functionalities, and other dynamic content.

JavaScript, jQuery

On the client-side, Javascript and jQuery are used both to handle nearly all the interactivity, animations, and effects in the system.

 
 

Project Structure

The project is basically a web application. All files for the web app will exist in the web directory depending on the web server. For XAMPP, this would be in the directory C:\xampp\htdocs\ChitChat, where ChitChat is the root folder.
