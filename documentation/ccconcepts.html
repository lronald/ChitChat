ChitChat Concepts

 

 

Back to index

 
 

Users

A user, as its name suggests, is a single person who owns an account with ChitChat. A user typically provides his/her username and password to provide them access to ChitChat functionalities & services.

 
 

Convos

A Convo is basically a chat session between users. This can be convos between two users, or group convos which involve more than two people.

 
 

Groups

The idea of groups in ChitChat is basically a collection of users which can converse together. Groups actually pave the way to allow for convos between multple users.

 
 

Messages

Messages refer to anything sent to a person or a group of people in ChitChat. Currently the sytem supports only text transmissions. However, future developments will incorporate other message forms.
